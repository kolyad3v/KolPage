A Record Keeping App along the Path

Using MongoDB I made an online record keeper that allowed me to note where I spent my time.

Allows users to log in and create their own "paths" permitting CRUD operations for a time/activity record of the user's choosing. Also comes with pre-built paths with custom entry fields relating to fitness and diet.

Utilises ThreeJS for more immersive dashboard experience.

Tech stack used: HTML/CSS - JavaScript - ReactJS - ExpressJS - MongoDB - NodeJS - ThreeJS - WebPack - GLSL - Blender
